# 🎓 CodeGuru – Online Learning Platform

Live Site: https://codeguru-project.netlify.app/

## 📌 Project Overview

CodeGuru is a full-stack online learning platform where instructors can add courses,
students can enroll, and admins can manage users and course approvals.

## 🚀 Key Features

- Firebase Authentication (Email & Google)
- Role-based Dashboard (Admin / Instructor / Student)
- Instructor Course CRUD (Add, Update, Delete)
- Admin Course Approval System
- Secure API with Firebase Token Verification
- Responsive UI with Tailwind CSS
- SweetAlert & Toast notifications
- Dark / Light mode ready
- MongoDB database

## 🧑‍💻 Tech Stack

- Frontend: React, Tailwind CSS, Axios, Firebase
- Backend: Node.js, Express.js, MongoDB
- Auth: Firebase Authentication
- Hosting: Firebase (Client), Vercel (Server)
